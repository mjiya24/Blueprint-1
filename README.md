# 🚀 Blueprint Architect

**Turn your skills, time, and situation into a real income plan—powered by AI.**

---

## 💡 What is Blueprint Architect?

Blueprint Architect is a mobile-first app that helps users **discover, plan, and execute personalized income strategies** based on their unique situation.

Instead of generic “side hustle ideas,” Blueprint generates **custom step-by-step blueprints** designed to help users go from **zero → first dollar → consistent income**.

---

## 🎯 The Problem

Most people:

* Don’t know where to start making extra income
* Get overwhelmed by too many generic ideas
* Lack structure, accountability, and direction
* Quit before seeing results

---

## ✅ The Solution

Blueprint Architect acts like a **personal income coach**, helping users:

* 🔍 Discover the best opportunities for *their* situation
* 🧠 Generate AI-powered, personalized income plans
* 📈 Track progress from idea → execution → earnings
* 🔁 Stay consistent with actionable steps and reminders
* 🏆 Celebrate real milestones (first client, first dollar, etc.)

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/mjiya24/Blueprint-1.git
cd Blueprint-1
```

### 2. Setup Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Setup Frontend

```bash
cd frontend
npm install
npx expo start
```

### 4. Environment Variables

**Backend**

```
MONGO_URI=your_mongodb_uri
STRIPE_SECRET_KEY=your_stripe_key
FIREBASE_CREDENTIALS=your_firebase_config
```

**Frontend**

```
EXPO_PUBLIC_API_URL=http://localhost:8000
```
